# ことのは かるた

ブラウザだけで遊べる、シンプルなかるたアプリです。

## 使い方

1. `開始`を押す
2. 表示された読み札に対応する取り札を選ぶ
3. 10問終わると結果が表示される

## GitHub Pages で公開する

このプロジェクトは静的ファイルのみなので、`main`ブランチ公開でOKです。

1. GitHubで新規リポジトリを作成（例: `karuta-app`）
2. ローカルでリモートを設定
3. `main`へpush
4. GitHubの `Settings` -> `Pages` で
   - `Build and deployment` -> `Source`: `Deploy from a branch`
   - `Branch`: `main` / `/ (root)` を選択

公開URLは通常以下になります。

`https://<your-github-id>.github.io/karuta-app/`
