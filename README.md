# 医薬品構造式かるた

医薬品の構造式（取り札）と説明文（読み札）を使ったフラッシュカード学習アプリです。

## 使い方

1. **カードをクリック / タップ** — 構造式 ↔ ヒント（説明文）を切り替えます  
2. **「覚えた」/ 「まだ」ボタン** — 学習進捗を記録します（ブラウザの localStorage に保存）  
3. **「未習得カードのみ表示」** — まだ覚えていないカードだけに絞り込みます  
4. **「シャッフル」** — カードの順番をランダムにします  
5. **「ヒントを先に表示」** — 表裏を反転します（ヒントを見て構造式を当てるモード）

### キーボードショートカット

| キー | 動作 |
|---|---|
| `→` / `↓` | 次のカード |
| `←` / `↑` | 前のカード |
| `Space` / `Enter` | カードを裏返す |
| `K` | 覚えた → 次へ |
| `U` | まだ → 次へ |

### スワイプ操作（スマートフォン）

カードを左右にスワイプして次/前のカードへ移動できます。

## ファイル構成

```
index.html          メインページ
style.css           スタイル
script.js           アプリロジック
data.json           カードデータ（構造体）
extract_cards.py    カード画像切り出しスクリプト（開発用）
assets/
  cards/
    structure/      構造式カード画像 (card-001.jpg 〜 card-050.jpg)
    hint/           ヒントカード画像 (card-001.jpg 〜 card-050.jpg)
  structures/       PDF から抽出したページ画像（原本）
```

## GitHub Pages への公開

1. このリポジトリを GitHub へプッシュ  
2. Settings → Pages → Source: `main` ブランチ / `/ (root)` を選択  
3. 保存後、数分で `https://<user>.github.io/<repo>/` で公開されます

`.nojekyll` ファイルにより Jekyll の処理をスキップしています。
